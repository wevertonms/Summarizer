    Summarizer.score_tokens(filterd_words, sentence_tokens)[source]¶

    Summarizer.summarize(ranks, sentences, length)[source]¶

    Utilizes a ranking map produced by score_token to extract the highest ranking sentences in order after converting from array to string.

    Summarizer.tokenize_content(content)[source]¶
    Accept the content and produce a list of tokenized sentences,
    a list of tokenized words, and then a list of the tokenized words with stop words built from NLTK corpus and Python string class filtered out.
